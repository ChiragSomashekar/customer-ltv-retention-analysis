# Interactive Dashboard (Tableau)

This project includes a single interactive Tableau dashboard with multiple interconnected views.

**Tableau Public link:**  
[View Dashboard](https://public.tableau.com/views/E-commerceProject_17666699417420/SalesDashboard?:language=en-GB&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

## Available dashboard views
- **Sales Dashboard** – revenue, orders, and AOV trends
- **RFM Dashboard** – customer segmentation based on recency, frequency, and monetary value
- **Funnels Dashboard** – user progression and drop-offs across key funnel stages
- **Cohorts Dashboard** – retention patterns and cohort-based lifetime value analysis

Navigation between views is handled via in-dashboard buttons and shared filters.
